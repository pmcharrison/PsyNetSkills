# PsyNet reference

The local PsyNet source checkout is expected at `~/PsyNet`.

Agents and contributors should inspect that checkout when they need concrete
PsyNet APIs, demos, testing commands, or implementation patterns. Prefer source
code and demos over guessing from memory.

Useful locations include:

- `~/PsyNet/demos/experiments/` for complete experiment examples.
- `~/PsyNet/demos/features/` for focused examples of individual PsyNet features.
- `~/PsyNet/docs/` for human-facing documentation.
- `~/PsyNet/psynet/resources/experiment_scripts/AGENTS.md` for agent setup and
  PsyNet command guidance.

PsyNet commands should be run from a suitable PsyNet checkout or experiment
directory. In Cursor, PsyNet command execution may need sandboxing disabled.
