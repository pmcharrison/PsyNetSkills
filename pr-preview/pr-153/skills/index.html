The currently implemented skills are listed below; see also the skills specification document.

Attempt a challenge

Attempt a PsyNetSkills challenge by creating a timestamped attempt folder, implementing the task, collecting evidence, and preparing outputs for human evaluation. Use when asked to attempt, run, or solve a challenge in this repository.

Source: .cursor/skills/attempt-challenge/SKILL.md

Create a challenge

Create a PsyNetSkills challenge from a prose task description, including public instructions, optional private criteria, references, validation, and dashboard-ready structure. Use when asked to draft, design, add, or create a challenge in this repository.

Source: .cursor/skills/create-challenge/SKILL.md

Create or update a skill

Create or update PsyNetSkills Agent Skills from prose lessons, recurring failures, attempt evaluations, or process improvements. Use when asked to add, revise, improve, or create a skill in this repository.

Source: .cursor/skills/create-skill/SKILL.md

Cursor cost estimation

Estimate and record Cursor agent run costs for PsyNetSkills challenge attempts from Cursor usage CSV exports.

Source: .cursor/skills/cursor-cost-estimation/SKILL.md

Dashboard preview polling

Poll the dashboard PR preview workflow before sharing preview links. Use when a Cursor Cloud Agent creates or updates a PR and needs to tell the user whether the dashboard preview reflects the latest pushed commit.

Source: .cursor/skills/dashboard-preview-polling/SKILL.md

Evaluate an attempt

Evaluate a completed PsyNetSkills challenge attempt conversationally, using the attempt evidence, copied criteria, user score, and user feedback to update EVALUATION.md. Use when asked to review, score, assess, or evaluate an attempt.

Source: .cursor/skills/evaluate-attempt/SKILL.md

Identify an author

Identify or register the human author for skills, challenges, and attempts before writing author metadata. Use when creating or updating author-attributed repository artifacts.

Source: .cursor/skills/identify-author/SKILL.md

Implement PsyNet experiments

Implement PsyNet experiments from text specifications, using local PsyNet source code, demos, setup commands, and validation practices. Use when asked to create, modify, test, or debug a PsyNet experiment implementation.

Source: .cursor/skills/psynet-experiment-implementation/SKILL.md

Implement real-time synchronous PsyNet experiments

Design and implement PsyNet websocket experiments with live synchronous interactions between multiple participants in the same trial.

Source: .cursor/skills/psynet-realtime-synchronous-experiments/SKILL.md

Make a PsyNet merge request

Create or verify merge requests against the upstream PsyNet GitLab repository from this workspace. Use when asked to send PsyNet changes upstream, check whether the agent has PsyNet MR privileges, or poll PsyNet CI for an MR.

Source: .cursor/skills/make-psynet-merge-request/SKILL.md

Prepare for translation

An experimental skill to prepare an existing PsyNet experiment for translation by marking participant-facing text, updating locale configuration, validating extraction, and committing the result.

Source: .cursor/skills/prepare-for-translation/SKILL.md

PsyNet Deployment Ops

Audit PsyNet experiment folders for deployment readiness, deployment records, export safety, app/server naming, EC2 teardown, and common local or SSH deployment blockers. Use when preparing PsyNet deployments, reconstructing deployment state, generating safe commands, or diagnosing deployment failures.

Source: .cursor/skills/psynet-deployment-ops/SKILL.md

Record participant video

Record a PsyNet participant flow to evidence/participant.mp4 with screen and experiment audio using ffmpeg. Use when collecting challenge evidence, creating participant.mp4, or documenting participant-facing behavior.

Source: .cursor/skills/record-participant-video/SKILL.md

Review skill overlap

Detect, prevent, and reorganize repeated guidance across PsyNetSkills Agent Skills before creating or updating a skill.

Source: .cursor/skills/skill-overlap-review/SKILL.md

Turn a pure-human experiment into an AI hybrid

Convert a pure-human PsyNet experiment into an AI or hybrid human-AI experiment with active scheduling, prompt parity, and local mock testing.

Source: .cursor/skills/turn-pure-experiment-to-ai-hybrid/SKILL.md

Verify AI model usability

Verify AI/LLM provider access, model availability, fallback options, and AI-call reliability for PsyNet experiments or user requests.

Source: .cursor/skills/verify-ai-model-usability/SKILL.md
