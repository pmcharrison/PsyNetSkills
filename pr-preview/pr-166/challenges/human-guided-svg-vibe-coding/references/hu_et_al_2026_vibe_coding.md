## Notes for challenge implementers

This challenge focuses on the paper's main human-led condition. Human
participants act as both instructors and selectors, while the AI system acts as
the SVG code-generation assistant. The full paper also reports AI-led, hybrid,
role-division, and independent evaluation experiments; those are useful context
but are not required for this challenge except for the independent similarity
rating task used to evaluate generated SVG outputs.

The main human-led condition used ten animal reference images, three repeated
chains per image, and 15 iterations per chain. Local challenge attempts may use
a smaller deterministic configuration, provided the implementation preserves
the same participant-facing roles and chain state.
