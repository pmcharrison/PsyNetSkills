# Analysis outputs

This directory was generated by `analysis.py`. If no export path is provided, the script uses a documented simulated dataset with one row per trial.

- `similarity_matrix.csv`: mean 0-6 ratings by stimulus pair.
- `discrimination_summary.csv`: same-different accuracy and reaction times.
- `identification_confusion.csv`: nearest-item correctness by probe, set size, and probe presence.
- `reaction_time_summary.csv`: reaction-time distributions by task block.
