# Ferry Market Unity WebGL build

Place the provided Unity WebGL build files in this directory when preparing a
challenge attempt or reference package.

Expected contents usually include:

- `index.html`
- `Build/`
- `TemplateData/`

If the build is too large to commit as individual files, add a `.zip` archive
under `challenges/ferry-market-unity/references/experiment/` instead; `.zip` files are
tracked with Git LFS in this repository.
