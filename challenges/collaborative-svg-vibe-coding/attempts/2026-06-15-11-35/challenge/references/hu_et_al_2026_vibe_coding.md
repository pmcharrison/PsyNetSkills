## Notes for challenge implementers

The challenge focuses on the paper's main experimental paradigm: iterative SVG
generation from reference animal images using instructor, selector,
code-generator, and evaluator roles. The implementation should prioritize the
participant-facing workflow and stored chain data over reproducing the full set
of paper analyses.
