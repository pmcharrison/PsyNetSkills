# Quorum personality and guessing game

This PsyNet experiment adapts the `sync_quorum` tutorial demo. Participants see
instructions, complete productive personality and guessing-game lobby trials
while a quorum forms, and then enter a tutorial-style quorate main loop once the
minimum active group size is present.

Run from this directory:

- `python experiment.py`
- `psynet test local`
- `psynet simulate`
