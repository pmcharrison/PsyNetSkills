# Learnings

## Public simulation instructions should expose scale and estimator expectations

The public instructions asked for a simulation demonstrating adaptive learning, while the hidden criteria required 30 adaptive participants, 30 non-adaptive participants, and an HMC accuracy comparison. The broader simulation was still implemented, but discovering this after first-pass evidence collection required reworking the analysis evidence.

*Actions:*
- **PsyNetSkills:** Update experiment implementation challenge public instructions when evaluator criteria require a specific simulation scale or estimator family, such as HMC, so agents can design evidence correctly before first-pass collection. Confidence: high. Impact: medium. Status: considering.
