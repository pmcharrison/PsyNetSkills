## Criteria

Ask the evaluator about each criterion and record the result here.

- [ ] Reaction time is measured using a minimal, isolated mechanism tied to
  PsyNet's native event-management system.
- [ ] Visual elements display with the specified timing, with no lingering
  fixation crosses, previous stimuli, probes, or unrelated graphical components.
- [ ] Numbered stimuli are clearly arranged around the fixation cross, with item
  numbers visually associated with the corresponding stimuli.
- [ ] The probe and response buttons are centered relative to the stimulus
  display.
- [ ] Participant-facing displays avoid technical labels or
  implementation-related text.
- [ ] PsyNet nodes and trial constructs are used correctly, and relevant trial
  metadata is recorded.
- [ ] Keyboard mapping occurs only during the response phase and maps correctly
  to numbered response options while preserving mouse clicks.
- [ ] Native PsyNet elements such as `KeyboardPushButtonControl`, `ModularPage`,
  and trial/node constructs are used appropriately.

## Notes

- Score and feedback should come from a human evaluator, captured
  conversationally when working with Cursor Cloud Agents.
