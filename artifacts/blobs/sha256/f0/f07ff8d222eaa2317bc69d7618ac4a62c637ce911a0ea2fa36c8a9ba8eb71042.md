# Evidence

This directory contains first-pass evidence for the completed attempt:

- `screenshots/`: Playwright screenshots for English, Italian, and Hebrew instruction, trial, validation, and debrief states.
- `participant.mp4`: short Hebrew participant walkthrough demonstrating RTL presentation and validation.
- `monitor.html`: local PsyNet dashboard snapshot from the evidence run.
- `performance.json`: `psynet performance-test local` JSON output.
- `simulated_data.zip`: combined English, Italian, and Hebrew `psynet simulate` exports.
- `simulated_exports/`: unzipped simulation exports used directly by the analysis notebook.
- `analyses/analysis.ipynb`: executed notebook reading the exported CSVs directly.
